# Changelog

## v0.1.0 beta

Initial beta release of AssetForge.

### Added

- Browser-based asset review workflow for Godot-style projects.
- Sample asset set for quick demos.
- Drag-and-drop, file picker, and folder import.
- Metadata detection for images, audio, lightweight model files, and data files.
- Godot-style `res://` path generation.
- Validation for unsupported formats, duplicate names, spaces, casing, snake_case, missing tags, and large files.
- Warning review controls for non-blocking issues.
- Search and filters by asset status and type.
- `manifest.json` and `import_notes.md` export previews and downloads.
- GitHub Pages deployment.
- Unit tests for metadata, validation, filtering, import, review, and export behavior.
